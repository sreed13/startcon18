What is Heroku?

A cloud platform based on a managed container system, with integrated data services and a powerful ecosystem, for deploying and running modern apps

How does it support fast application development?

It's designed by developers for developers

We can link a GitHub repository and use a command line interface and simple user interface to support continuous integration

We can use Heroku Pipelines to deploy through a continuous delivery workflow using review, staging and production applications

How does scale?

Heroku runs on Amazon Web Services (AWS) and can scale horizontally and vertically to support enterprise workloads

How does AI fit in?

Heroku has an ecosystem of add-ons that are fully managed

Einstein Vision and Einstein Language are 2 examples of add-ons that developers can use to build AI powered apps fast

Einstein Vision allows you to build AI-powered applications by embedding image recognition into your application workflows so that you can do things like visual product search, product identification, and automated planogram analysis. Einstein Vision includes two APIs: Einstein Image Classification and Einstein Object Detection.

Einstein Language allows you to build AI-powered applications by embedding natural language processing into your application workflows so that you can do things like intelligent case routing, community sentiment analysis, and smart lead trending.
